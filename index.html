用 hexo 快速搭建博客（一）——写在前面

发表于 2017-12-02

第一次用 hexo 是我刚开始学编程的时候。

那时我是菜鸟，是因为看到一个好看的主题，叫 yilia 就想要做一个这样的 blog。结果我一看，很复杂的样子，它不是那种 download 下来，一行代码就能跑的。

那个时候我认为，一个简单的项目应该是 download 到本地，然后输入一行代码就可以运行起来的。

所以我觉得这个东西不够简单，对新手还不够友好，当然也可能是我基础不好…

如今我再来看，这个工具用起来挺方便，很快就可以上手，但是 hexo 官网说明对新手来说，还是不够简单。并且很少有讲怎么放到 GitHub pages 上（虽然这步更加简单）。

所以我准备写一个简单的小教程，想象的教学对象是当初刚学编程的我，目标是基础不好的小白也可以看懂。。。
